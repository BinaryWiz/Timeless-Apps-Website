Price Assist

Price Assist wants to save people time. We solved the problem with you tediously spending your precious time sifting through many retailers to find the lowest prices.Instead, install Price Assist and watch the magic as it automatically finds the best price for your electronic devices.

A Couple of Useful Projects

UPM

UPM, or Unsupervised Product Matching, is an algorithm created for Price Assist that attempts to classify products from different retailers as the same product or not. Although very ambitious, it seems to have potential. If you want to check it out, click the button below.

UPM GitHub

LevelDB Caching Server

The LevelDB Caching Server is a project that allows a server to store a JSON response in a LevelDB database for a set amount of time. This essentially allows a user to set an expiration date for each cached item. This is very useful for servers where data is constantly changing. Check it out below.

Caching Server GitHub
